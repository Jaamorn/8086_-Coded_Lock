# 一个基于 STAR ES598PCI 8086实验系统的密码锁
##概述
* 这是一个可定时更换密码的密码锁
* 汇编代码基于星研集成环境
* 控制器代码基于恩智浦mk60dn512zvlq10单片机
* 使用一个74HC08芯片作为脉冲转换模块
* 电路原理图使用Altium Designer绘制
## 目录说明
| 文件/目录        | 说明           |
| :----------- | :----------- |
| Controler    | 密码控制器/显示器源代码 |
| Circuit      | 电路原理图        |
